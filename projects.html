Statsample

What is it about?

Statsample-Timeseries is extension of Statsample, and incorporates helpful time series functions, estimation techniques, and modules, such as:

  • ACF
  • PACF
  • ARIMA
  • Kalman Filter
  • Log Likelihood
  • Autocovariances
  • Moving Averages

Statsample-Timeseries is part of the SciRuby project.

Instamojo Ruby

What is it about?

This is the Ruby library of Instamojo REST API. This will assist you to programmatically create, edit and delete links on Instamojo. Also supports RAP api for payments request, listing and status.

I started this as an independent project, it was later officially adopted by Instamojo.

Uber Ruby

What is it about?

This is the Ruby interface of Uber Developers API. This enables developers to integrate Uber's Drivers, Rides, Activities etc in their applications and workflows.

I am the co-author of this project along with Dingding Ye of GrowingIO. This library was officially adopted by Uber.

Duss

What is it about?

Duss is an open-source URL shortener written in GoLang. It works with any SQL variant.

It supports configuration of key length, url normalization and collisions handling. More features of duss are in the works.
